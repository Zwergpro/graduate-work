Веб-приложение для работы с рекомендательными моделями

Данное веб-приложение предоставляет инструменты для создания моделей рекомендательных систем.


Основные возможности
  • Сбор данных для построеня индекса врачей
  • Сбор данных для обучения моделей CatBoost
  • Сбор данных для тестирования
  • Тренировка моделей и хранение результатов
  • Тестирование моделей и хранение результатов
  • Отслеживание процессов в реальном времени

На странице создания датасетов меется функционал для параметризации собираемых данных.

На странице тренировки моделей есть возможность указать параметры которые будут учитываться при тренировке.

На странице тестирования можно проверить эффективность разработанных ранее моделей.
